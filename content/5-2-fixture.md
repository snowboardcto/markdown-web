# Markdown Formatting Sampler

This page demonstrates a handful of common Markdown elements as they render on The Markdown Web.

## Headings and Text

Use `#` for headings. Body text renders in a comfortable reading measure with GitHub-style typography.

## Lists

- Plain text files — no proprietary format
- Works offline, syncs on push
- Every page is a real URL

## Code

Inline `code` and fenced blocks are highlighted at build time, so they look great without any client-side JavaScript.

```md
# Hello, Markdown Web

Write here. Push to publish.
```

Add any `.md` file to the vault and it appears on the site after the next build.
