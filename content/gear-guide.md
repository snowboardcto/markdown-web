# Gear Guide

The canonical Story 2.3 link target. A relative link `[guide](gear-guide.md)`
from another page must resolve to this page's route `/gear-guide`.

It links back to the [home](x.md) page (resolves to `/x`) and to the
[nested](sub/page.md) page (resolves to `/sub/page`) so the round-trip plus
browser back/forward (AC5) is exercisable.
