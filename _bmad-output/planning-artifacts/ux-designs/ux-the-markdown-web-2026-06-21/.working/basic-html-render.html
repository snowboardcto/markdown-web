Powder Day Gear Guide

A markdown file, rendered. Same .md, basic HTML for any browser.

Riding deep snow is its own discipline. This guide covers the essential setup for a backcountry powder day — what to bring, what to leave, and the one thing you should never forget. Prefer the raw source? It's just powder-day.md.

Avalanche safety first. No line is worth your life. Check the forecast, carry the gear, and never ride alone.

The checklist

  • Beacon, shovel, probe (and know how to use them)
  • Powder board or a stiffer all-mountain deck
  • Spare goggles — fogging ends days early
  • Snacks that don't freeze solid

Board sizing

Float comes from surface area. As a rough rule, size up in powder:

Tuning the setback

Most powder boards ship with a reference stance. To set a deeper centered back-foot bias, shift the inserts and re-check your binding angles:

# shift stance 30mm back for float
def setback(stance_mm, shift=30):
    return {
        "front": stance_mm.front - shift,
        "back":  stance_mm.back  - shift,
    }

A quick note on wax

  1. Cold snow wants a hard, cold-temp wax.
  2. Iron, scrape, brush — in that order.
  3. Re-wax more often than you think; powder is abrasive.
