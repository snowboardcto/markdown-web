### Theming approach ŌĆö PINNED (the contract for Step-4 TDD + Step-5 impl)

**Decision: extend `FlowDocumentRenderOptions` with the theme tokens + apply them in `FlowDocumentRenderer`'s existing mappers ŌĆö NOT a separate post-render theme applier.** Rationale:
- The renderer already builds every element with its 3.3/3.4 markers; the cleanest additive change is to set the additional style properties (`Margin`/`LineHeight`/`Background`/`BorderBrush`/`Foreground`/`Padding`/`MarkerOffset`) on those same elements as they are built, reading the values from `_options`.
